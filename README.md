# Fire Palace Ordering System

火宫殿点餐系统 - 餐厅智能点餐管理平台

## 项目简介

Fire Palace Ordering System 是一个现代化的餐厅点餐系统，提供用户点餐、服务员管理、包厢分配等功能。

## 技术栈

- **后端**: Spring Boot + MyBatis + MySQL
- **前端**: React Native + Expo + NativeWind

## 项目结构

```
├── src/
│   ├── backend/          # Spring Boot 后端
│   └── RN_frontend/      # React Native 前端
├── doc/                  # 文档
└── API文档.md            # API接口文档
```

## 快速开始

### 后端

```bash
cd src/backend
mvn spring-boot:run
```

### 前端

```bash
cd src/RN_frontend
npm install
npm start
```
